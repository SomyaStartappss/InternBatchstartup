# Urban Somya – Fashion Website

This is a responsive fashion website named **Urban Somya** built using **HTML** and **CSS**, showcasing products for Men, Women, Kids, and Gen Z audiences. The website is designed to provide an attractive, user-friendly interface to explore fashion trends, product collections, and offers.

---

## Project Overview

**Urban Somya** focuses on delivering a seamless browsing experience with features like:

- A responsive header with navigation menu.
- Interactive sections highlighting products, collections, and special offers.
- Customer engagement section for queries and contact.
- Visually appealing hero section with images and promotional content.
- Footer with links, social icons, and user interaction forms.

The website uses **flexbox** and **CSS grid** for layout and media queries for responsiveness.

---

## Technologies Used

- **HTML** – Structure of the website, including header, hero section, product highlights, blog-style collections, and footer.
- **CSS** – Styling for layout, colors, gradients, shadows, fonts, hover effects, and responsive behavior.
- **Font Awesome** – For icons such as social media, menu toggle, and interactive elements.

---

## Key Features

1. **Responsive Design**  
   - Adapts to desktop, tablet, and mobile screens.
   - Sidebar navigation on small screens with toggle buttons.

2. **Hero Section**  
   - Large banner with call-to-action button.
   - Split layout with text on the left and product image on the right.

3. **Feature Boxes**  
   - Highlights reasons to shop: latest fashion trends, quality products, fast delivery, and style for everyone.
   - Grid layout with hover animation for interactivity.

4. **Detail Section**  
   - Promotional offer with input form to place orders.
   - Split layout with text and form on the left, image on the right.

5. **Blog/Collection Section**  
   - Displays collections for Men, Women, and Gen Z.
   - Each collection is shown in card-style layout with hover effects.

6. **Footer**  
   - Contact information, social media icons, and call-to-action buttons.
   - Links to user agreement, legal info, and partner programs.

7. **Responsive Sidebar**  
   - On smaller screens, navigation moves to a sidebar with open/close toggle.

---

## How It Works

- **Header & Navigation:** The header contains the website logo and a navigation menu. On smaller devices, the menu collapses into a sidebar controlled by checkbox toggles.
- **Hero Section:** Eye-catching section with main tagline, description, and shop button. Uses flexbox for layout.
- **Product/Feature Boxes:** Grid layout displaying main selling points with images and text. Hover effect slightly lifts boxes for interactivity.
- **Detail Section:** Encourages users to place an order using a form. Input fields are styled for better UX.
- **Blog Section:** Displays collections with images and descriptions in card layout.
- **Footer:** Divided into contact info, social icons, and links. Maintains consistent styling with the theme.
- **Responsive Design:** Media queries adjust layout for tablets and mobile screens, hide or show elements as needed for optimal UX.

## Usage

1. Open the project folder in VS Code.
2. Open the `index.html` file in a browser.
3. Navigate through sections using header links or scroll down.
4. Fill in forms in the detail or footer sections to test interactivity.
5. Resize the browser to see responsive behavior.
