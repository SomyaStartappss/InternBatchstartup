# Simple Calculator App

This project is a basic web-based calculator built using **HTML, CSS, and JavaScript**. It allows users to perform simple arithmetic operations such as addition, subtraction, multiplication, and division.

---

## Project Description

The calculator app provides a user-friendly interface for performing calculations. Users can enter numbers and operators, view the current expression, and get the result immediately.

**Features:**

- Input numbers and operators directly via buttons.
- Supports basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Clear button to reset the display.
- Handles division by zero gracefully.
- Displays an error message for invalid input or expressions.
- Responsive design for easy use on various screen sizes.

---

## Technologies Used

- **HTML**: Defines the structure of the calculator including buttons and input display.
- **CSS**: Styles the layout, colors, spacing, and responsive alignment of the calculator.
- **JavaScript**: Implements the logic for arithmetic operations, input handling, and display updates.

---

## How It Works

1. **User Input**: Users click on the number and operator buttons to form an expression.
2. **Display**: The current expression is shown in the input display at the top.
3. **Calculation**:
   - When the "=" button is pressed, JavaScript evaluates the expression.
   - If the expression matches the required format (number → operator → number), it calculates the result.
   - For division, dividing by zero displays "Error".
   - Invalid inputs or unsupported expressions display "Invalid".
4. **Clear Function**: Clicking the "Clr" button clears the display to start a new calculation.

---

## Usage

1. Open the project in a web browser.
2. Use the buttons to enter numbers and arithmetic operators.
3. Click "=" to calculate the result.
4. Click "Clr" to clear the display and start over.
5. The calculator only supports single-step operations (e.g., `5+3`).
